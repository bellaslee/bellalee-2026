# Core aesthetic

Think:

- soft porcelain white
- blue-gray ink
- restrained detail
- lots of breathing room
- subtle ornamental influence, not literal grandma china wallpaper

The vibe should feel quiet, refined, editorial, with a little handcrafted
character.

# Color palette

Use a very tight palette.

Base

- Porcelain: #F8F7F3
- Warm ivory: #F2EFE8

Primary ink

- Cobalt blue: #355C7D
- Deep indigo ink: #213548

Secondary

- Mist blue: #D8E3EA
- Dusty slate: #8FA3B5

Accent

- Soft gold-gray: #B9B1A3

How to use it:

- Background: porcelain / warm ivory
- Headings: deep indigo
- Buttons, links, dividers: cobalt blue
- Cards and subtle fills: mist blue at low opacity
- Accent linework: soft gold-gray very sparingly

Do not overdo the blue. The porcelain effect comes more from restraint +
contrast + spacing than from making everything blue.

# Fonts

- Headings: Cormorant Garamond
- Body/UI: Inter

## Type scale:

- H1: 64–80 px
- H2: 36–44 px
- H3: 24–28 px
- Body: 16–18 px
- Small/meta: 13–14 px
