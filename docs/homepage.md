# Homepage Structure

## Top Navigation

- Left: Name / personal logo
- Right: Resume · Roadmap · Tools · Projects · About

---

## Hero Section

**Layout:** Two-column (asymmetrical)

**Left column**

- Eyebrow label (e.g., “Software Engineer”)
- Large headline (name + concise value statement)
- Short intro paragraph (what you build / what problems you like)
- Primary CTA: View Resume
- Secondary CTA: Explore Projects

**Right column**

- Subtle visual panel:
  - porcelain-inspired line motif OR
  - “Currently exploring” focus card OR
  - abstract framed shape

---

## Identity Strip

Slim horizontal section with 3–4 descriptors:

- Developer tools
- Interactive systems
- Learning in public
- Product-minded engineering

---

## Main Feature Cards

Asymmetrical layout:

- Left: Large featured card — **Interactive Resume**
- Right: Two stacked cards
  - Learning Roadmap
  - Developer Tools

Each card contains:

- Title
- One-sentence description
- Small bullet list of contents
- Subtle link / arrow CTA

---

## Featured Project Section

Two-column layout:

- Left:
  - Project title
  - Short problem statement
  - Tech stack tags
  - Key outcome or insight
  - Link to full case study
- Right:
  - Project screenshot OR architecture visual

---

## Current Learning Preview

Compact roadmap snapshot:

- 2–3 current topics
- Progress indicators
- Links to notes / demos

---

## Footer

- Email
- GitHub
- LinkedIn
- Short personal note or tagline
